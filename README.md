# ai-for-the-arts-b
This notebook will bridge the gap between the 19th and 21st centuries to bring to life the illustrations presented in Andrew MacGeorge’s Travel Journal of a Continental Tour (1867-1868). 

The purpose of this notebook is to produce a product which incorporates generative AI and the human creativity processes jointly. The output will engage audiences to add value to enhance cultural heritage of the product and introduce new perspective yet also exploring social implications introduced throughout processes. The result being a project that augments creativity when utilizing AI as a tool. 

This will be done by completing the following objectives :

1.	Digitization of the object to create a dialogue between the historical object digital tool to increase audience engagement. 
2.	Use a machine learning application as a speculative reconstruction tool to distinguish and employ components of generative models to create historically informed colourization of sketches.  
3.	Demonstrate creative processes through human-ai collaborations to convey the objects story through sentiment and emotion to employ a modern twist on a 19th century object 
4.	Address any social concerns relating to Ai-driven creative content to evaluate how Ai usage shapes cultural heritage.
